Frequently Asked Questions

How does the symptom heatmap work?

Premium feature showing color-coded visualization of when symptoms occur across cycle phases. Helps identify patterns in logged data.

How does the visual calendar work?

Calendar displays logged period data with color-coded indicators on each date. Flow intensity shown with different shades: pink for spotting/light, red for medium/heavy. Ovulation window appears as purple gradient when enabled.

How are estimates calculated?

Estimates based on logged cycle history. Ovulation estimates typically use 14-day window before expected period. Fertile window spans approximately 5 days. These are estimates based on patterns and may vary.

How do I restore premium purchase?

Tap "Unlock Full History" then "Restore Purchase". Use same Apple ID from original purchase. Purchases managed through Apple's system.

Is my data private?

Yes. All data stored locally on device only. Clara never uploads data to servers or shares with third parties.

What does free version show?

Free version displays last 2 months of logged data. All data continues being stored safely on device. Premium unlocks access to data beyond 2 months and symptom heatmap.
